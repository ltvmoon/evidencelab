## Research Trees

Research Trees let you drill down into topics by building an explorable tree of sub-queries from search results.

### How It Works

1. **Start from search results** — run a search and view the AI summary
2. **Drill down** — highlight text in an AI summary or click "Find out more" on a top heading to automatically generate a sub-query
3. **Context inheritance** — each sub-query inherits the root search query plus the immediate parent topic for context, building deeper understanding
4. **Navigate the tree** — use the tree view to navigate back to any previous node to review its results and summary
5. **Filter inheritance** — sub-queries also inherit all active filters (data source, date range, etc.) from the parent search

### Use Cases

* **Literature review** — start with a broad topic and systematically drill into specific aspects
* **Gap analysis** — explore related sub-topics to find where evidence is strong or lacking
* **Evidence synthesis** — build a structured overview of a topic by exploring multiple branches
